### Q15 形态信号回测——早晨之星的胜率

> 过去 30 个交易日中，每天出现"早晨之星"(pattern_morning_star)的股票，次日涨跌幅的平均值和胜率分别是多少？

```python
dates = d.list_available_dates()
recent = dates[-30:]
panel = d.load_range(recent[0], recent[-1])

signals = panel[panel.pattern_morning_star == 1].copy()
next_ret = panel.groupby("ts_code").apply(
    lambda g: g.set_index("trade_date")["pct_chg"].shift(-1)
).reset_index()
next_ret.columns = ["ts_code", "trade_date", "next_pct_chg"]

signals = signals.merge(next_ret, on=["ts_code", "trade_date"])
signals = signals.dropna(subset=["next_pct_chg"])

avg_ret = signals.next_pct_chg.mean()
win_rate = (signals.next_pct_chg > 0).mean()
print(f"信号数量: {len(signals)}")
print(f"次日平均涨跌幅: {avg_ret:.2f}%")
print(f"胜率: {win_rate * 100:.1f}%")
```

### Q16 多因子综合评分选股

> 在 2026年5月20日的截面上，用 PE（越低越好）、换手率（越高越好）、主力净流入（越大越好）三个因子做排名打分，加权（各占 1/3）选出综合得分前 20 的股票。

```python
subset = day.dropna(subset=["pe", "turnover_rate", "net_mf_amount"]).copy()
subset["pe_score"] = subset.pe.rank(ascending=True)
subset["turnover_score"] = subset.turnover_rate.rank(ascending=False)
subset["mf_score"] = subset.net_mf_amount.rank(ascending=False)
subset["total_score"] = (subset.pe_score + subset.turnover_score + subset.mf_score) / 3
subset.nsmallest(20, "total_score")[
    ["ts_code", "name", "pe", "turnover_rate", "net_mf_amount", "total_score"]
]
```

### Q17 板块轮动——行业资金流随时间变化

> 过去 20 个交易日，每天各行业的主力净流入总额分别是多少？找出持续 3 天以上净流入为正的行业。

```python
dates = d.list_available_dates()
recent = dates[-20:]
panel = d.load_range(recent[0], recent[-1])

ind_flow = panel.groupby(["trade_date", "industry"])["net_mf_amount"].sum().reset_index()

pivot = ind_flow.pivot_table(index="trade_date", columns="industry", values="net_mf_amount")
positive_days = (pivot > 0).sum()
continuous = positive_days[positive_days >= 3]
print(f"持续 >=3 天净流入为正的行业:\n{continuous.sort_values(ascending=False)}")
```

### Q19 涨停晋级率分析

> 统计最近 10 个交易日的首板(pattern_first_limit)股票中，次日有多少继续涨停（命中 first_limit 或 multi_limit）？计算每日晋级率。

```python
dates = d.list_available_dates()
recent = dates[-11:]
panel = d.load_range(recent[0], recent[-1])

first_limit = panel[panel.pattern_first_limit == 1][["ts_code", "trade_date"]].copy()

panel["is_limit_next"] = (
    (panel.pattern_first_limit == 1) | (panel.pattern_multi_limit == 1)
).astype(int)

next_day = panel.groupby("ts_code").apply(
    lambda g: g.set_index("trade_date")["is_limit_next"].shift(-1)
).reset_index()
next_day.columns = ["ts_code", "trade_date", "next_is_limit"]

merged = first_limit.merge(next_day, on=["ts_code", "trade_date"]).dropna(subset=["next_is_limit"])

promo = merged.groupby("trade_date").agg(
    first_count=("ts_code", "count"),
    promoted=("next_is_limit", "sum"),
)
promo["rate"] = promo.promoted / promo.first_count * 100
promo
```
