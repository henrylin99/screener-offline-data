# screener_daily 大宽表数据字典 / Data Dictionary

> 本字典基于**最新全市场快照** `data.parquet`（交易日 `20260519`）实测生成，反映当前 canonical 列结构。历史分区目录 `data/screener_daily/daily/year=*/month=*/day=*/data.parquet` 使用同一套列结构，逐日拼接即面板数据。

## 文件概览 / File Overview

| 项目 | 说明 |
|---|---|
| 文件类型 | Parquet 日线宽表快照 |
| 所属表 | `screener_daily` 选股器日线大宽表 |
| 行数 | `5484`（每行 = 1 只股票当日截面） |
| 列数 | `240` |
| 逻辑主键 | `ts_code + trade_date` |
| ts_code / name 唯一值 | 各 `5484`（全市场单日无重复） |
| industry 唯一值 | `110` |
| area 唯一值 | `32` |

## 结构分组 / Schema Groups

| 分组 | 字段数 | 说明 |
|---|---:|---|
| 基础标识 | 5 | 证券身份与分区信息 |
| 行情 OHLC 与成交 | 11 | 开高低收、涨跌、量、额、换手、量比 |
| 估值与市值 | 16 | 估值倍数、股息、股本、市值 |
| 资金流 | 18 | 分单级别买卖量额 + 净流入 |
| 复权价 | 11 | 前/后复权价 + 复权因子 |
| 技术指标 | 14 | MACD / KDJ / RSI / BOLL / CCI |
| 财务（`fin_`） | 33 | 利润表、资产负债表、现金流量表 + 衍生比率 |
| 动量因子 | 8 | 新高突破、连涨、5 日量比 |
| 形态信号（`pattern_`） | 125 | 预计算 K 线形态 0/1 标志 |
| 其它 | 1 | `pct_change` 冗余涨跌幅 |

## 1. 基础标识 / Identity

| 字段 | 类型 | 说明 |
|---|---|---|
| `ts_code` | str | 证券代码，如 `000001.SZ` |
| `name` | str | 证券简称，如 `平安银行` |
| `industry` | str | 所属行业（110 类） |
| `area` | str | 所属地区（32 类） |
| `trade_date` | str | 交易日 `YYYYMMDD` |

## 2. 行情 OHLC 与成交 / Quote & Volume

| 字段 | 类型 | 单位 | 说明 |
|---|---|---|---|
| `open` `high` `low` `close` | float | 元 | 当日开/高/低/收盘价（**未复权**） |
| `pre_close` | float | 元 | 前收盘价 |
| `change` | float | 元 | 涨跌额 |
| `pct_chg` | float | % | 涨跌幅（推荐用此列） |
| `vol` | float | 手 | 成交量 |
| `amount` | float | 元 | 成交额 |
| `turnover_rate` | float | % | 换手率（总股本） |
| `turnover_rate_f` | float | % | 换手率（自由流通股本） |
| `volume_ratio` | float | 倍 | 量比 |

> `pct_change` 是冗余的涨跌幅列（有 29 个缺失），分析统一用 `pct_chg`。

## 3. 估值与市值 / Valuation & Market Cap

| 字段 | 类型 | 单位 | 说明 |
|---|---|---|---|
| `pe` `pe_ttm` | float | 倍 | 市盈率 / 滚动市盈率 |
| `pb` | float | 倍 | 市净率 |
| `ps` `ps_ttm` | float | 倍 | 市销率 / 滚动市销率 |
| `dv_ratio` `dv_ttm` | float | % | 股息率 / 滚动股息率 |
| `total_share` `float_share` `free_share` | float | 万股 | 总股本 / 流通股本 / 自由流通股本 |
| `total_mv` | float | 万元 | 总市值 |
| `circ_mv` | float | 万元 | 流通市值 |

## 4. 资金流 / Money Flow

> 单位：`*_vol` 为**手**，`*_amount` 为**千元**。买卖按单笔金额分 4 档：sm 小单 / md 中单 / lg 大单 / elg 特大单。

| 字段族 | 说明 |
|---|---|
| `buy_sm_vol` `buy_sm_amount` `sell_sm_vol` `sell_sm_amount` | 小单买/卖 量、额 |
| `buy_md_*` `sell_md_*` | 中单买/卖 量、额 |
| `buy_lg_*` `sell_lg_*` | 大单买/卖 量、额 |
| `buy_elg_*` `sell_elg_*` | 特大单买/卖 量、额 |
| `net_mf_vol` | 主力净流入量（手） |
| `net_mf_amount` | 主力净流入额（**千元**，可正可负） |

主力净买入常用 `buy_lg_amount + buy_elg_amount − sell_lg_amount − sell_elg_amount` 构造。

## 5. 复权价 / Adjusted Prices

> **回测算收益必须用复权价**，原始 `close` 跨除权日不连续。

| 字段 | 类型 | 说明 |
|---|---|---|
| `adj_factor` | float | 复权因子 |
| `open_hfq` `high_hfq` `low_hfq` `close_hfq` `pre_close_hfq` | float | 后复权 OHLC + 前收 |
| `open_qfq` `high_qfq` `low_qfq` `close_qfq` `pre_close_qfq` | float | 前复权 OHLC + 前收 |

前复权（`_qfq`）适合看近期，后复权（`_hfq`）适合长周期连续收益计算；同一序列内保持一致即可。

## 使用约定 / Conventions

- **算收益用复权价** `*_qfq` / `*_hfq`，不要用原始 `close`。
- **金额单位**：`amount` 元、`total_mv`/`circ_mv` 万元、资金流 `*_amount` 千元、`fin_*` 元。换算前先核对本表。
- **`pattern_*` / `break_high_*` / `consec_up_3/5` 是 0/1**；`consec_up_days` 是计数；其余多为连续值。
- **跨日对齐**用 `(ts_code, trade_date)`；`download_history` 返回的就是这样的面板数据。
- **缺失值**：绝大多数列无缺失，少数财务比率与 `pct_change` 有零星 NaN，聚合前按需 `dropna()`。
